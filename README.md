# Paradise Nursery Shopping Application

## Project Overview
Paradise Nursery is an e-commerce web application for an online plant shop. The application allows users to browse a variety of houseplants, view plant details, add plants to their shopping cart, and manage cart items with a dynamic and interactive user interface.

## Features
- **Landing Page**: Welcome page with company information and a "Get Started" button
- **Product Catalog**: Browse plants organized by categories (Air Purifying, Aromatic, Insect Repellent, Medicinal, Low Maintenance)
- **Shopping Cart**: Add, remove, and adjust quantities of plants in the cart
- **Dynamic Cart Updates**: Real-time cart count and total cost calculations
- **Responsive Design**: Modern UI with smooth transitions and interactive elements

## Technologies Used
- React.js
- Redux Toolkit (State Management)
- Vite (Build Tool)
- CSS3
